# EnTerapia.EA — Sitio Web de Ericka Acosta

Este es el sitio web de **Ericka Acosta, M.A**, licenciada en Psicología Clínica y **neuropsicóloga infantil**, con formación en lenguaje, conducta y educación inclusiva.
